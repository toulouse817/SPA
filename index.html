Sobre el Proyecto

Pila Tecnológica Vanilla

Explorando características modernas sin sobrecarga. Una pila ligera y de rendimiento súper rápido.

Enrutador JavaScript Vanilla

Cuenta con un enrutador de hash del lado del cliente que escucha los cambios en la URL. El cambio dinámico del DOM ocurre instantáneamente sin recargas completas de la página, utilizando sistemas de eventos nativos.

CSS de Glassmorphism Premium

Diseñado estrictamente utilizando propiedades personalizadas de CSS3 (variables) para las transiciones de tema. Mejorado con motores de diseño avanzados como CSS Grid, Flexbox, filtros de fondo (backdrop) y microanimaciones de fotogramas clave de tipo resorte.

Integración con la API Random User

Integrado con funciones asíncronas de JavaScript async/await. Implementa estados de carga estructurados, alternativas en caso de error, renderizado de cuadrícula adaptativo, filtros de búsqueda y vistas de detalles en ventanas emergentes (modales).
