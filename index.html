About the Project

Vanilla Technical Stack

Exploring modern features without overhead. A lightweight, super-fast performance stack.

Vanilla Javascript Router

Features a client-side Hash Router listening to URL states. Dynamic DOM switching happens instantaneously without full page reloads, using native event systems.

Premium Glassmorphism CSS

Styled strictly using CSS3 Custom Properties (variables) for theme transitions. Enhanced with advanced layout engines like CSS Grid, Flexbox, backdrop filters, and spring-like keyframe micro-animations.

Random User API Integration

Integrated with asynchronous JavaScript async/await functions. Implements structured loading states, error fallbacks, responsive grid renderings, search filters, and detail views in modals.
